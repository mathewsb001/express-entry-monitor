# Express Entry Monitor

Automated bot that monitors the IRCC Express Entry page and sends notifications when new draws are detected.

## Features

- Monitors Express Entry draws every 10 minutes
- - Sends email notifications when new draws are detected
  - - Generates Canva posters with draw information
    - - Integrates with Zapier for additional automation
